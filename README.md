# Replication Code for "Sensitivity analysis for contamination in egocentric-network randomized trials"

This repository contains the code to replicate the data analysis and 
simulation studies from the paper "Sensitivity analysis for contamination in egocentric-network randomized trials" (Weinstein & Nevo, 2025).

The methodological code (sensitivity analysis framework) is available as a separate R package, `ENRTsensitivity`.

## R Package Installation

Before running the replication scripts, you must install the `ENRTsensitivity` package from GitHub:

```r
# install.packages("devtools")
devtools::install_github("barwein/ENRTsensitivity")
```
